	-Interaction = Information Interchange
	-Interaction needs a communication medium
	-Programming Language is used as the Communication Medium

ASCII:
	American Standard Code for Information Interchange
	- Assigns an individual decimal number for all the keys in the keyboard
	Ex:
	ASCII Value for 	A = 65
			a = 97
			0 = 48
